xhtml2doc

NAME

xhtml2doc - generate manual pages or Texinfo sections from a restricted XHTML source.

SYNOPSIS

USAGE

gawk [options] -f xhtml2man xhtml-file
gawk [options] -f xhtml2texi xhtml-file

DESCRIPTION

The xhtml2man and xhtml2texi awk scripts can generate both troff-formatted manual pages and/or texi-formatted document sections from a specialized xhtml source file. Options are passed as -v option=value arguments in the command line.

Common options:

OUTFILE=file
Name of the generated file. Default is the same name of the source with the .xhtml suffix changed either to the man section name or to .texi.

Options for the xhtml2texi conversion only:

TOPLEVEL=(chapter/section)
The Texinfo section level generated by the <h1> tag. Default "section".
NUMBERLEVEL=(0..4)
The maximum <hn> tag that generates numbered sections in the Texinfo document. Default 2.
NODELEVEL=(0..4)
The maximum <hn> tag that generates nodes in the Info document. Default 2.
MENULEVEL=(0..4)
The maximum <hn> tag that generates automatic menus in the Info document. Default 2.

In all cases the effective maximum numbered or node heading is "subsubsection"

xhtml2man specific HTML elements

Required elements are marked as **. Sample values correspond to this manpage.

  • ** <title>XHTML2DOC</title>
    (short title, in caps)
  • ** <meta name="section" content="3am"/>
    (the manpage section: 1, 2, ...)
  • ** <meta name="date" content="February 2025"/>
    (date of the last nontrivial update)
  • ** <meta name="source" content="GAWK Extension Library (gawkextlib)"/>
    (the global entity that provides this manpage)
  • ** <meta name="manual" content="GNU Awk Extension Modules"/>
    (the documentation set to which the manpage belongs)
  • -- <h1>xhtml2doc</h1>
    (ignored, useful only for conversion to Texi)
  • ** <h2>NAME</h2><p>xhtml2doc - generate manual pages ...</p>
    (first main section, with one-line short description of the utility, prepended with the utility name in lowercase and a dash)
  • -- <h2>SYNOPSIS</h2>
    (other main sections: SYNOPSIS, DESCRIPTION, NOTES, AUTHORS, ...)
  • -- <hx>Conditional elements</hx>
    (x=3,4,..: subsections)

xhtml2texi specific HTML elements

Required elements are marked as **. Elements marked as *- are only required if the document section is in fact a manpage to be formatted as Texinfo. Sample values correspond to this manpage.

  • ** <h1>xhtml2doc</h1>
    (title of the section or chapter)
  • *- <h2>NAME</h2><p>xhtml2doc - generate manual pages ...</p>
    (NAME section, with one-line short description of the utility, prepended with the utility name in lowercase and a dash)
  • -- <hx>Conditional elements</hx>
    (x=2,3,..: other sections, indexed by the whole title)
  • -- <hx title="index-term">...</hx>
    (x=2,3,..: other sections, indexed by the index-term)
  • -- <dfn>defined-term</dfn>
    (defined-term indexed as such)
  • -- <dfn title="index-term">defined-term</dfn>
    (defined-term indexed by the index-term)

General supported XHTML elements

The XHTML source files can use the following tags to generate contents in the <body> part.

<div>      block fragment
<span>     inline fragment
<dl>       definition list
<dt>       definition term
<dd>       definition paragraph
<ol>       numbered list
<ul>       bulleted list 
<li>       list item
<pre>      preformatted text
<br>       line break
<em>       italic
<i>        italic
<var>      italic
<b>        boldface
<strong>   boldface
<code>     monospace
<kbd>      monospace
<samp>     monospace
<tt>       monospace

Partially supported, and only with xhtml2texi:

<table>    table
<col>      column specification
<tr>       table row
<th>       table header
<td>       table cell

Conditional elements

The same XHTML source can be used to generate both a manpage and a Texinfo section. The class attribute of any tag can be used to process the element only in one alternative.

  • <xx class="man" ...>...</xx> (processed only by xhtml2man)
  • <xx class="texi" ...>...</xx> (processed only by xhtml2texi)

Tags with a class attribute with values other than man or texi are fully ignored, i.e., neither the tag nor its content is processed.

Unsupported elements

Elements not listed before are not supported. The tag itself is ignored but its content is processed.

LIMITATIONS

<dfn> inside <hn> section titles is ignored.

<div> and <span> are only useful as conditional elements.
